Ringkasan Hasil

Eksperimen menunjukkan bahwa:

- Penjumlahan sinyal meningkatkan amplitudo total.
- Penggeseran mengubah posisi sinyal dan citra.
- Amplifikasi mengubah amplitudo sinyal dan brightness citra.
- Sistem T(x)=2x memenuhi homogenitas dan additivitas.
- Sistem T(x)=x² tidak memenuhi sifat sistem linier.

Kesimpulan

Operasi dasar seperti penjumlahan, penggeseran, dan amplifikasi merupakan fondasi penting dalam pengolahan sinyal digital dan pengolahan citra.
